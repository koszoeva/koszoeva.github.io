Players

get /players
Returns the list of players in the database.

Request

  • playerId (optional)
    The ID of the player.
    Type: String
    Example: 12345
  • name (optional)
    The name of the player.
    Type: String
    Example: 67890
  • country (optional)
    The home country of the player.
    Type: String
    Example: Scotland

Response

200 - Success

Example data

[ {
  "country" : "Scotland",
  "name" : "Adrian Honker",
  "playerId" : "12345",
  "nicknames" : [ "The Feather Shot", "The Snookering Swan" ],
  "bestRanking" : 8,
  "currentRanking" : 10
}, {
  "country" : "Scotland",
  "name" : "Bob Potts",
  "playerId" : "12321",
  "nicknames" : [ "The Potmeister", "Magic Bob" ],
  "bestRanking" : 39,
  "currentRanking" : 41
} ]

post /matches
Creates a new match.

Request

  • matchId (required)
    The ID of the match.
    Type: String
    Example: 112233
  • tournamentId (required)
    The ID of the tournament.
    Type: String
    Example: 67890
  • player1Id (required)
    The ID of player 1.
    Type: String
    Example: 12345
  • player2Id (required)
    The ID of player 2.
    Type: String
    Example: 54321
  • score (required)
    The result of the match.
    Type: String
    Example: 5-3
  • referee (optional)
    The name of the match referee.
    Type: String
    Example: Max Fouls
  • refereeCountry (optional)
    The home country of the match referee.
    Type: String
    Example: Wales
  • date (required)
    The date of the match.
    Type: Date
    Example: Thu Apr 20 00:00:00 GMT 2023

Response

201 - Created

Example data

{
  "date" : "2023-04-20T00:00:00.000+00:00",
  "player1Id" : "12345",
  "score" : "5-3",
  "refereeCountry" : "Wales",
  "player2Id" : "54321",
  "tournamentId" : "67890",
  "matchId" : "112233",
  "referee" : "Max Fouls"
}

post /tournaments
Creates a new tournament.

Request

  • tournamentId (optional)
    The ID of the tournament.
    Type: String
    Example: 67890
  • name (optional)
    The name os the tournament.
    Type: String
    Example: World Championship
  • location (optional)
    The location of the tournament.
    Type: String
    Example: Sheffield, UK
  • isRanking (optional)
    True if the tournament is a ranking event; false otherwise.
    Type: Boolean
    Example: 10
  • isInvitational (optional)
    True if the tournament is an invitational event; false otherwise.
    Type: Boolean
    Example: Sat Apr 15 00:00:00 GMT 2023
  • startDate (optional)
    The start date of the tournament.
    Type: Date
    Example: Sun Apr 30 00:00:00 GMT 2023

Response

201 - Created

Example data

{
  "name" : "Masters",
  "tournamentId" : "555666",
  "location" : "London, UK",
  "isInvitational" : true,
  "isRanking" : false,
  "startDate" : "2023-01-07T00:00:00.000+00:00"
  "endDate" : "2022-01-14T00:00:00.000+00:00",
}
